# Contributing

Thanks for your interest in contributing! 🎉  
This project is beginner-friendly and open to new contributors.

## How to contribute

1. Fork the repository
2. Create a new branch
3. Pick an issue or suggest your own
4. Make changes and commit
5. Open a Pull Request
 
## Good first issues

Look for issues labeled:
- good first issue
- help wanted

## Tech stack

- React
- Vite
- Firebase
- SCSS
- Framer Motion

## Code style

- Clear and readable code
- Meaningful commit messages
- Small, focused PRs

## Communication

Feel free to open an issue or ask questions in PR comments.
