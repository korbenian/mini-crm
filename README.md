# 🧠 Mini CRM / Productivity Dashboard

A multifunctional SPA for task management, reading frontend-related articles, and communicating with AI.  
This project was created as a pet project to practice a modern frontend stack and SPA architecture.

## ✨ Features

- 🔐 **Authentication**
  - Login page
  - Registration page
- 📊 **Dashboard**
  - Main page with quick access to all features
- 📝 **Tasks**
  - Create tasks
  - Filter and sort tasks
  - Manage task lists
- 📚 **Frontend Articles**
  - Dedicated page with educational frontend content
- 🤖 **AI Chat**
  - Communicate with AI directly inside the application
- 👤 **User Profile**
  - View profile
  - Edit profile information
- 🌍 **Multilingual Support**
  - English
  - Russian
  - German
  - Dutch
  - Polish
  - French
- 🌗 **Themes**
  - Light theme
  - Dark theme

## 🛠️ Tech Stack

- **Frontend**
  - React
  - TypeScript
  - Vite
- **Architecture**
  - Single Page Application (SPA)
  - Component-based architecture
- **Additional**
  - i18n (multilingual support)
  - API integration
  - Serverless functions (Vercel)

## 🚀 Running the Project

### The AI chat functionality is implemented using Vercel serverless functions.

To use the AI chat, the project must be started via vercel dev.

Run with AI Support
npm install -g vercel
vercel de
### Standard Run (without AI chat)

```bash
npm install
npm run dev
